# 🚀 Projeto Hello World - Spring Boot

Este repositório contém um projeto simples desenvolvido em **Spring Boot** como parte das atividades da **Generation Brasil**.  
O objetivo é praticar a criação de **endpoints REST** e entender melhor a estrutura de um projeto Spring.

## 🛠️ Tecnologias Utilizadas

- **Java Spring Tools for Eclipse**
- **Spring Boot**
- **Maven**
- **Collections Framework**
- **Git/GitHub**
